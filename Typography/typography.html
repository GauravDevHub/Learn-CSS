Understanding Typography in CSS

What is Typography in CSS?

Typography in CSS refers to the style, appearance, and arrangement of text within a web page. It plays an important role in enhancing readability, accessibility, and the overall user experience. CSS offers various properties to control text size, spacing, line height, font family, and more.

1. Font Family

The font-family property allows you to define the font for the text. You can specify a list of fonts for the browser to try in order.

This text uses Arial as the font family.

If Arial is unavailable, the browser will try sans-serif.

2. Font Size

The font-size property controls the size of the text. You can use values in various units like pixels (px), em, rem, percentage, etc.

This text has a font size of 24px.
This text has a font size of 2em, which is relative to the parent element.

3. Font Weight

The font-weight property allows you to control the thickness of the text. Common values include normal, bold, and numeric values like 100, 400, and 700.

This text has a bold font weight.
This text has a 100 (light) font weight.

4. Line Height

The line-height property controls the amount of space between lines of text. It's important for readability, especially in paragraphs.

This text has a line height of 2, meaning the lines are twice as spaced.

5. Text Alignment

The text-align property is used to align text horizontally within its containing element. Common values are left, center, right, and justify.

This text is centered using the text-align: center property.
This text is justified using the text-align: justify property. It stretches the text to fit the container.

6. Letter Spacing

The letter-spacing property controls the space between characters in the text. It's helpful for creating effects or improving readability.

This text has a letter spacing of 2px.

7. Text Transform

The text-transform property allows you to control the capitalization of text. Values include uppercase, lowercase, and capitalize.

This text is in uppercase.
this text has capitalized first letters for each word.
