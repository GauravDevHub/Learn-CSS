What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation (look and feel) of a document written in HTML or XML. CSS controls the layout, design, colors, fonts, and overall appearance of a webpage. It enables web developers to create visually appealing and consistent user interfaces.

1. Inline CSS

Inline CSS is used to apply styles directly to HTML elements using the style attribute. It affects only that specific element and is useful for quick styling of single elements.

This is an example of Inline CSS. The background color is applied directly using the style attribute.

2. Internal CSS

Internal CSS is written within the <style> tag in the <head> section of an HTML document. It is used to style the entire page and affects all elements within that document.

This is an example of Internal CSS. The styling is applied from the <style> tag in the <head> section.

3. External CSS

External CSS is written in a separate .css file and linked to the HTML document using the <link> tag. This method is ideal for styling multiple pages consistently across a website.

This is an example of External CSS. The styling comes from an external stylesheet (not shown here, but linked via the <link> tag).
