Types of CSS Selectors

What is a CSS Selector?

CSS Selectors are patterns used to select and style HTML elements. With selectors, you can define which elements you want to apply your styles to. There are several types of selectors based on how they target the elements.

1. Universal Selector

The universal selector * targets all elements in the document. It is useful for applying global styles.

This text is styled by the * (universal) selector, which applies a global font family to all elements.

2. Type Selector

A type selector targets all elements of a specific type (tag name). For example, p targets all paragraph elements.

This text is styled by the p (type) selector, affecting all paragraph elements.

3. Class Selector

This text is styled by the .highlight (class) selector, targeting elements with the class highlight.

4. ID Selector

The ID selector #idname targets a specific element with the given ID. It is unique and can be applied only once per page.

This text is styled by the #unique (ID) selector, targeting the element with the ID unique.

5. Attribute Selector

Attribute selectors target elements based on their attributes. For example, input[type="text"] targets all text input fields.

6. Pseudo-classes

Pseudo-classes target elements in specific states, such as :hover for when an element is hovered over.

7. Pseudo-elements

Pseudo-elements allow you to style a specific part of an element, such as the first letter or the first line. For example, ::first-letter can be used to style the first letter of a paragraph.
