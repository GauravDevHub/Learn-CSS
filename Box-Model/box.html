CSS Box Model Explained

What is the CSS Box Model?

The CSS box model is a fundamental concept that defines how elements are displayed and how space is calculated on a webpage. Every element is considered as a rectangular box made up of:

  • Content - The actual content of the element (text, image, etc.)
  • Padding - Space between the content and the border
  • Border - A border that wraps the content and padding
  • Margin - Space outside the border, separating the element from others
Tip: Total element width = content + padding + border + margin. Use box-sizing: border-box to include padding and border inside the width.
