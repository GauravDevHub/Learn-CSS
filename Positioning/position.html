CSS Position Properties Explained with Live Examples

Understand how different CSS position values work with visual demonstrations and simple explanations.

1. position: static (default)

Static Box

Static is the default position for all elements. It follows the normal document flow and doesn't respond to top/left/right/bottom values.

2. position: relative

Relative Box

Relative positioning shifts the element from its normal position, but it still occupies its original space in the flow.

3. position: absolute

Absolute Box

This absolute box is placed inside a relative container. It is positioned using top/right/left/bottom values from the nearest ancestor that is not static.

4. position: fixed

Fixed Box

Fixed elements stay in place relative to the viewport, even when scrolling the page. This is often used for sticky navigation or action buttons.

5. position: sticky

Sticky Box

Sticky positioning toggles between relative and fixed depending on scroll position. It sticks to the top when scrolling past it.
